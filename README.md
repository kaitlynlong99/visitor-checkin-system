# Cloud-Based Visitor Check-In System
A web application that allows organizations to manage and track visitor check-ins. 
The system records visitor information and stores it in a database, simulating a real cloud-hosted environment. 

## Technologies Used
- Python
- Flask
- SQL
- HTML / CSS

## Features
- Web interface for visitor check-ins
- Backend logic built with Flask
- Data stored and retrieved using SQL
- Administrator dashboard to view checked-in visitors
- Secure administrator login with hashed password authentication

## Security
Administrator login credentials are protected using password hashing to securely store authentication data.

## Project Purpose
This project was created to practice full-stack development concepts, including backend programming, database interaction, and deploying applications in a cloud environment. 

## How It Works
1. A visitor enters their information through the web interface.
2. The application processes the data using Flask.
3. Visitor records are stored in a SQL database.
4. An administrator can log in to view the dashboard of checked-in visitors.
